## Accessibility Features

- **Keyboard Navigation**: Tab through sections, Enter to expand/collapse
- **Screen Reader Labels**: Descriptive ARIA labels for all interactive elements
- **Color + Icon**: Never rely on color alone (each priority has unique icon)
- **Hover Tooltips**: Full account names visible on hover (for truncated text)
- **Focus Indicators**: Clear visual focus states for keyboard users

## Performance Optimizations

- **Lazy Rendering**: Only render expanded sections
- **Virtual Scrolling**: For accounts list (if > 20 accounts)
- **Memoization**: useMemo for expensive calculations
- **Debounced Expansion**: Prevent rapid expand/collapse
- **Query Caching**: React Query cache (5 min stale time)
